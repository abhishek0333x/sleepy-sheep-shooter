# 🐑 Sleepy Sheep Shooter 🌙

A whimsical nighttime shooting game where sleepy sheep wander across a moonlit meadow. Click them before they escape!

## Features

- 🎯 **30 shots** — make every click count
- ⚡ **Speed ramp** — sheep get faster over time
- 🔥 **Combo system** — rapid hits multiply your score
- ✨ **Beautiful night scene** — twinkling stars, glowing moon, fireflies & animated grass

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- **React** + **Vite**
- Pure SVG sheep (no image assets)
- CSS keyframe animations
- RequestAnimationFrame game loop
